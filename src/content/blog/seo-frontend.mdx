# SEO и современный frontend

Frontend сегодня влияет не только
на интерфейс, но и на поисковую
видимость проекта.

## Почему это важно

- скорость загрузки
- semantic HTML
- accessibility
- structured data
- mobile optimization

## Next.js и SEO

Next.js предоставляет:

- SSR
- static generation
- metadata API
- routing architecture

Это делает framework сильным
решением для SEO-ориентированных
проектов.
